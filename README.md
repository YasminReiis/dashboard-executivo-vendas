# 📊 Dashboard Executivo de Vendas

Projeto de análise de dados desenvolvido para simular um cenário corporativo de monitoramento de performance de vendas através de indicadores estratégicos.

O objetivo é demonstrar como dados podem ser transformados em informações visuais e insights que auxiliam gestores na tomada de decisões.

---

# 🎯 Objetivo do Projeto

Empresas utilizam dashboards executivos para acompanhar a performance comercial e identificar oportunidades de crescimento.

Este projeto simula a construção de um dashboard estratégico para análise de vendas, permitindo responder perguntas importantes como:

- Qual canal gera mais receita?
- Quais produtos têm maior impacto nas vendas?
- Como as vendas evoluem ao longo do tempo?
- Existe sazonalidade nas vendas?

---

# 📊 Indicadores de Negócio (KPIs)

Os principais indicadores analisados no projeto incluem:

| KPI | Descrição |
|----|----|
| Receita Total | Valor total gerado em vendas |
| Total de Vendas | Quantidade total de pedidos realizados |
| Ticket Médio | Valor médio gasto por compra |
| Receita por Canal | Performance de vendas por canal (Online, Loja Física, etc.) |
| Receita por Produto | Identificação dos produtos com maior geração de receita |
| Evolução Mensal | Análise da tendência de vendas ao longo dos meses |

---

# 📈 Análises Realizadas

Durante o projeto foram realizadas análises exploratórias e estratégicas para identificar padrões relevantes nos dados:

- Identificação dos canais de venda mais lucrativos
- Análise dos produtos com maior contribuição para a receita
- Avaliação da evolução mensal das vendas
- Identificação de possíveis padrões de sazonalidade

Essas análises permitem compreender melhor o comportamento das vendas e apoiar decisões estratégicas.

---

# 📊 Resultados e Insights

A análise dos dados permitiu identificar alguns pontos importantes:

- Determinados canais concentram a maior parte da receita da empresa
- Alguns produtos apresentam forte impacto no faturamento total
- Existe variação significativa nas vendas ao longo dos meses, indicando possíveis padrões sazonais
- O ticket médio permite avaliar o comportamento de consumo dos clientes

Esses insights podem ajudar gestores a otimizar estratégias de vendas, marketing e gestão de produtos.
